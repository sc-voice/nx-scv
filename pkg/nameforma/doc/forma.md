# Forma

## Overview

- Base class for identifiable named objects
- Tracks instance counts by prefix
- Supports patching (merging) properties
- Validation of UUID v7 and name prefixes

## Avro Schema

```
{
  name: 'Forma',
  namespace: 'scvoice.nameforma',
  type: 'record',
  fields: [
    { name: 'id', type: 'string' },              // immutable, unique UUID64
    { name: 'name', type: 'string' },            // mutable
  ]
}
```

## Features

1. **Named Identity**
   - Extends Identifiable with mutable `name` property
   - Default name format: `{PREFIX}-{UUID_PREFIX}`
   - Example: `T2K-01234567` for Task instances

2. **Instance Tracking**
   - Static `#instances` map tracks count per class prefix
   - Prefix generated by abbreviating class name
   - Example: "Task" → "T2K" (first + count + last)

3. **Patching**
   - `patch(cfg)` method merges properties
   - Only updates mutable fields (name)
   - Immutable id preserved

4. **Validation**
   - `validate(opts)` checks UUID v7 format
   - Optional name prefix validation
   - Returns `true` or `Error`
